# Product Overview

OptScale (branded as CloudHub) is an open-source FinOps and cloud cost optimization platform. It helps organizations manage, monitor, and optimize their cloud infrastructure spending across multiple cloud providers (AWS, Azure, Google Cloud, Alibaba Cloud, Kubernetes).

## Core Capabilities

- Cloud cost tracking, breakdown, and expense analysis across providers
- Resource management with assignment rules, pools, and budgets
- Cost optimization recommendations (rightsizing, reserved instances, savings plans, duplicate detection)
- ML/AI profiling: task tracking, run management, datasets, models, metrics, leaderboards, and runset templates
- Environment and booking management for shared resources
- Power scheduling for instance start/stop automation
- Business Intelligence export
- Organization-level policies, constraints, and alerts
- Integrations: Slack, Jira, Google Calendar, webhooks
- Live demo generation for prospects
- Multi-cloud data import and processing pipeline

## Product Naming

- The internal project name is "OptScale"
- The user-facing brand is "CloudHub" (see URLs like `cloudhub.com`, `my.cloudhub.com`)
- The company behind it is Hystax
